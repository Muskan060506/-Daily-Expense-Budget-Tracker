#  Daily Expense & Budget Tracker

##  Description
The **Daily Expense & Budget Tracker** is a Java-based application that helps users record daily expenses, track spending, and manage their monthly budget efficiently.


##  Features
- Add daily expenses with category, amount, and date  
- View all recorded expenses  
- Calculate total expenditure  
- Set and monitor monthly budget  
- Get alerts when budget exceeds  

##  Tech Stack
- Java  
- Object-Oriented Programming (OOP)  
- Collections (ArrayList)  
- File Handling

##  How to Run
1. Clone the repository  
2. Open in any Java IDE (IntelliJ, Eclipse, VS Code)  
3. Compile all `.java` files  
4. Run `Main.java`  
5. Use the menu options in the console
